***CREATE PROJECT***
1. 
Create Virtual Environment & Install Django
cd /path/to/dev/folder
mkdir try_django
cd try_django
pipenv install django
pipenv shell

2.
Create Django Project
cd /path/to/dev/folder
mkdir src
cd src
django-admin startproject try_django
